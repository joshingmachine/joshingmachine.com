Keyboard

# Link to "keyboard" section
Logitech ERGO K860 Wireless Split Keyboard

I had been using the Apple Magic Keyboard because it came with my work iMac, but my wrist started hurting and I decided to look into ergonomic keyboards. The K860 has been great, the pain disappeared almost immediately after switching to it.

# Link to "mouse" section

Browsers

# Link to "browsers" section
Firefox

I've used Chrome, Safari, Opera, etc., and am comfortable with them, but Firefox is my jam. Chrome is a little too memory intensive for my Thinkpad, and thus I made the switch to Firefox. Also, I get the feel-goods by supporting browser diversity. I've really enjoyed using it as my primary browser, and just started using Firefox Focus as my default browser on iOS, so we'll see how that goes.
